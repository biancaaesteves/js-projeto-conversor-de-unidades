### PROJETO CONVERSOR DE UNIDADES DE MEDIDAS

Neste projeto, adicionei as unidades padrão de medida de comprimento no Sistema Internacional de Unidades (SI). 

Aqui estão elas em ordem decrescente de magnitude:

- Quilômetro (KM)
- Hectômetro (HM)
- Decâmetro (DAM)
- Metro (M)
- Decímetro (DM)
- Centímetro (CM)
- Milímetro (MM)

## Tecnologias:

- HTML
- CSS
- JavaScript.
